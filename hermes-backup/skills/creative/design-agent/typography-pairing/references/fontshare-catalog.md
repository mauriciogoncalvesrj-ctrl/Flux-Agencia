## How to Use This Catalog

1. **Find by personality**: Read character descriptions. Match your brand voice.
2. **Get CDN link**: Copy the URL. Paste into `<link>` or `@import`.
3. **Check weights**: Use only the weights you need (fewer = smaller file size).
4. **Pair strategically**: Combine display + body font using the six pairings as reference.

---

## Quick Pairing Rules

- **Display**: Pick from Clash Display, Satoshi, Cabinet Grotesk, Zodiak, Boska, or alternatives
- **Body**: Pick from General Sans, Inter, Switzer, Panchang, or Erode
- **Contrast**: If display is geometric, body can be geometric (similar) or humanist (contrast)
- **Personality match**: Display personality should match brand. Body should always prioritize readability.
