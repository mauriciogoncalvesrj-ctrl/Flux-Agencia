## Como usar este diretorio

### Para consultar uma pesquisa
1. Abra o arquivo pelo codigo ou data
2. Cada pesquisa contem: resumo executivo, concorrentes mapeados, persona, gaps, oportunidades e campanhas prontas

### Para iniciar uma nova pesquisa
1. Use a skill `/market-research` no Claude Code
2. Ou siga o template em `template-pesquisa.md`
3. Salve o resultado com padrao de nome: `YYYY-MM-DD-<codigo>-<tema>.md`

---

## Padrao de nomenclatura

- **Arquivo:** `YYYY-MM-DD-CODIGO-tema.md`
- **Codigo:** `SETOR-SEQ` (ex: B2B-EST-001, B2B-ODON-002)
- **SETOR:** B2B, B2C, ECOS, COMP
- **SEQ:** Sequencial numerico por setor
