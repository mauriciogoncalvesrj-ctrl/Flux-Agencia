Estilo visual aprovado para a Agência Flux, baseado em carrosséis de referência enviados em 09/05/2026.

**Why:** Mauricio quer consistência visual em todo o conteúdo gerado. Sem essa referência, outputs ficam genéricos e fora do padrão.

**How to apply:** Antes de gerar qualquer copy, prompt de design ou brief de criativo para a Flux, consultar o arquivo `.agents/estilo-visual-referencia.md` no projeto.

## Resumo do Estilo

- **Paleta:** Preto + Dourado + Branco (autoridade) | Preto + Azul elétrico (tech/IA) | Preto + Vermelho (urgência)
- **Tipografia:** Ultra bold, caixa alta, títulos gigantes que dominam o slide
- **Tom visual:** Dark, premium, impactante — transmite autoridade e urgência
- **Backgrounds:** Preto sólido (Claude Design) ou fotorrealistas com atmosfera (Midjourney/Flux)

## Tipos de Slide Aprovados

1. **Capa de impacto** — título enorme + subtítulo colorido + fundo dramático
2. **Lista com ícones** — fundo preto + itens com ícones dourados/azuis
3. **Dado/prova** — número grande em destaque + contexto menor
4. **Foto do fundador** — Mauricio com overlay tech/IA
5. **CTA** — instrução direta + ícone de ação
